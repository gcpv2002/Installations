# Installing Apache on Linux
## To install Apache on Linux, follow the steps below:
Navigate to the directory in which you would like to install Apache using `cd` command

To update the system, use the command
```
sudo apt update && sudo apt upgrade -y
```

To install Apache, use the command
```
sudo apt install apache2 -y
```

To check the version, use the command
```
apache2 -v
```
