# Installing Certbot for Apache on Linux
## To install Certbot and Apache plugin on Linux, follow the steps below:

To update the system, use the command
```
sudo apt update && sudo apt upgrade -y
```

To install Certbot and Apache plugin, use the command
```
sudo apt install certbot python3-certbot-apache -y
```

To check the version, use the command
```
certbot --version
```
